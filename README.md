# AgenOptimic

AgenOptimic là một hệ thống multi-agent được thiết kế để giải quyết các tác vụ tối ưu hoá tổ hợp. Dự án sử dụng LangGraph(state managment) để điều phối các tác nhân chuyên biệt như Coder, Planner và Formulator nhằm giải quyết bài toán tối ưu hoá tổ hợp với input là bài toán biểu diễn dưới dạng ngôn ngữ tự nhiên.
